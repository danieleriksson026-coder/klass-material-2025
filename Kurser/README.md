# 📚 Kursmaterial

Här organiseras allt material per kurs/ämne.

## Struktur per kurs:

```
📁 [Kursnamn]/
  📁 Vecka-01/
  📁 Vecka-02/
  📁 Föreläsningar/
  📁 Prov-och-tentor/
  📁 Labbar/
  📄 README.md (kursöversikt)
```

## Exempel på kurser:
- Matematik
- Svenska  
- Historia
- Naturkunskap
- Engelska

**Skapa nya mappar genom att klicka "Create new file" och skriva `Kursnamn/README.md`**
